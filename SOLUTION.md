# JS & Rockets
Provided solution renders JSON format list of missions from 2018 that carried vast of payload owned by NASA.

## Getting Started

### Dependencies
Application is powered by Apollo Server, Apollo Client and GraphQL for building a data graph and managing communication layers, apollo server testing and Jest test libraries. The structure of project contains of directory for server configuration separated from the interface development. Tests for the server integration and coverage overwiew are located under /src/server from the root. Versions of npm 6.13.7 and node -v 12.16.0

### Installing

Install dependencies

```
npm install
```

Install dependencies for the server

```
cd src/server & npm install
```

Building the project
```
../../ & npm run build
```
Starting the project
```
npm run start
```

## Running the tests

```
npm run test
```
Tests --watchAll
```
npm run test-watch
```
Coverage and reports
```
npm run test-coverage
```
